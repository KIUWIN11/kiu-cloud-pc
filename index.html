// How to get a GitHub Token
Get Your GitHub Token
1
Go to GitHub Token Settings
Click below — it opens GitHub with the codespace scope already checked.

👉 github.com/settings/tokens/new →
2
Sign in to GitHub
If not already signed in, GitHub will prompt you to log in first.
3
Set Expiration (Optional)
Choose how long the token lasts — 30 days, 90 days, or No expiration.
4
Click "Generate token"
The codespace permission is already ticked. Scroll to the bottom and click the green "Generate token" button.
5
Copy Your Token
Your token starts with ghp_.... Copy it immediately.
⚠️ The token is only shown once. If you close the page, you'll need to generate a new one.
6
Paste into the Token Field Above
Go back to Create Cloud PC, paste your token, select a machine, and hit "Create Cloud PC". Done! 🎉
